# Gerenciador de Usuários - Python 🐍

![Python](https://img.shields.io/badge/Python-3.x-blue)
![License](https://img.shields.io/badge/License-MIT-green)

Mini sistema em Python para gerenciar usuários usando JSON.  
Permite:

- Listar usuários  
- Contar usuários ativos/inativos  
- Buscar usuários por nome  
- Adicionar novos usuários com ID e status aleatório  

---

## Funcionalidades

1. **Listar usuários**: exibe todos os usuários cadastrados.  
2. **Contar usuários ativos/inativos**: mostra estatísticas do arquivo JSON.  
3. **Buscar usuário por nome**: pesquisa ignorando maiúsculas, minúsculas e espaços.  
4. **Adicionar novo usuário**: cria usuário com ID único, nome, email e status ativo/inativo aleatório.  
5. **Sair do programa**: encerra a execução de forma limpa.
